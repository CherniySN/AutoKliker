Данная программа была разработана как временное решение для старого ПО Ак Барс банка. 
Прграмма выполняет обновления экрана посредством зажатия клавишь. 
Программа представляет собой десктопное приложение.

161 нажатие клавишь за 4 секунды. 
Кливиша F1 нажимается каждые 0,02 секунды.
Каждые 4 секунды происходит пауза между нажатиями в 0,3 секунды.
